# 🏥 Dr. Shiral Hospital Website

A modern and responsive hospital website developed for **Dr. Shiral Hospital** using **HTML, CSS, and JavaScript**. The project focuses on providing an intuitive user experience, responsive design, and easy access to hospital information and services.

## 🚀 Features

* Responsive and mobile-friendly design
* Modern user interface
* Hospital information and services section
* Contact and inquiry section
* Interactive navigation and user experience
* Optimized front-end performance

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📁 Project Structure

drshiral-website/
├── index.html
├── styles.css
└── main.js

## 🎯 Project Objective

The objective of this project is to create a professional and user-friendly hospital website that improves accessibility to healthcare information and enhances the overall user experience.
